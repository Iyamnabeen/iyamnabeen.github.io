Here are the programs i use daily. I only list programs i consider tried and have used for a while.

Basic

  • Operating system

I use Arch Linux, because it requires you to build up your own ‘distribution’ from the ground up.

  • Terminal

I use st(simple terminal) by suckless, which is one of the most minimal, yet easily customizable terminal emulator out there.

  • Window Manager

herbstluftwm is a manual tiling window manager for the X window system. It can be configured in many ways(with a shell script, or even with python too!), so it is very extensible.

  • Shell

I use zsh as a shell. It has pretty much all of the features of bash with additional plugins for more advanced auto-completion and syntax highlighting.

  • Text editing

vim. Less of a text editor and more of a lifestyle. Technically i use neovim nowadays but its all the same.

  • Web Browser

Brave. Probably the only browser that doesnt requrie significiant addition of plugins to make it functional. Blocks ads and trackers by default and comes with anti-fingerprinting abilities and Tor windows.
